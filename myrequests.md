我有一个关于cloud code的提示词（cloud.md）。这个提示词用于帮助学生通过学习和考试，像一个tutor或指导者来辅导学生。这个提示词对我意义特别大，我希望把它改成英语学习的claude.md。比如说我平时看书的时候，会发现一些看不懂的英文句子，原因可能是有不认识的单词、不熟悉的词组，或者语法问题。我想重新写这个提示词，当我打开这个claude code项目后，可以发送一个段落，询问不明白的地方，让它用类似的方式给我讲明白。讲明白之后，我们之间的对话可以形成一个session文档。 

类似:

```

/learn-notes/

  2025-10-11.md

  2025-10-12.md

```

我的复习时间分为三个阶段：早晨复习、晚上复习和周末复习。晚上复习时，我会把一天当中不会的单词和语法都看一遍。为了验证自己是否真的理解了这些单词的意思，我会看着这些英文自己验证一下是否搞懂了。为了提高学习效率，晚上复习时最好把当天所有的单词形成一篇简短的文章让我来复习。早晨复习主要是复习昨天晚上的那篇文章。周末复习则是复习一周的内容。

复习方式都是一样的：选择一个时间范围内的单词、词组和语法，生成一篇英文文章，看我能否读懂。如果读懂了，就算这些单词、语法或词组我已经掌握了。如果没懂，我会重新把它记录到当天的列表中。

生成文章的需求我会封装成一个(claude)command命令(会选择指定的时间范围)，这个命令会让你生成文章。生成文章时，要形成这样的目录：

```

/review-article/
  2025-10-11.md  

```

生成文章后让我阅读。如果我有看不懂的地方，会通过对话告诉你。这些没看懂的内容，同样需要记录在/learn-notes/目录下，算作当天不会的单词和内容。这样每天或每周复习起来会比较方便，也不用额外准备特殊的文章了
